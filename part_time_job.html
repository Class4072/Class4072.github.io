아르바이트 리뷰

----------------------------

아르바이트 이름 : 아이스링크장 스태프

업무 : 매표, 스케이트 대여, 안전요원

장점 : 어린 손님들이 즐겁게 노는걸 보면 신이 난다. / 스케이트를 무제한으로 탈 수 있다.
단점 : 컴플레인의 강도가 매우 높다. / 추위를 많이 타는 사람은 매우 힘들다.

추천도 : ★★★☆☆


아르바이트 이름 : 부대찌개집 아르바이트

업무 : 홀서빙, 주방보조

장점 : 직원식이 맛있다. / 브레이크 타임때는 쉬면서 돈을 번다.
단점 : 취객 진상을 봐야한다. / 위생 관리가 힘들다.

추천도 : ★★★☆☆


아르바이트 이름 : 회사 업무보조

업무 : 업무보조

장점 : 회사생활을 경험할 수 있다.
단점 : 회식문화도 배워야 한다.

추천도 : ★★★★☆


아르바이트 이름 : 공장 조립

업무 : 부품 조립

장점 : 시급이 쎄다.
단점 : 지루하다.

추천도 : ★★★★☆


아르바이트 이름 : 김치 공장

업무 : 재료 옮기기, 재료 손질하기

장점 : 시중에 있는 HACCP인증 제품들을 신뢰할 수 있다.
단점 : 정말 힘들다.

추천도 : ☆☆☆☆☆


아르바이트 이름 : 편의점 아르바이트

업무 : 접객, 물류 정리

장점 : 신상품들을 누구보다 빠르게 만나 볼 수 있다. / 일이 편하고 쉽다.
단점 : 점포마다 일의 강도가 다르다.

추천도 : ★★★★★
